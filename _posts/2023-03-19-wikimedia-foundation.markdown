# бһўбһ„бҹ’бһӮбһ—бһ¶бһ– Wikimedia (WMF)

бһўбһ„бҹ’бһӮбһ—бһ¶бһ– WikimediaвҖӢ бһҮбһ¶бһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһҳбһ·бһ“бһҡбһҖбһҖбһҳбҹ’бһҡбҹғбһҳбһҪбһҷбһҡбһ”бһҹбҹӢбһўбһ¶бһҳбҹҒбһҡбһ·бһҖ бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһ“бһ¶бһҖбҹӢбһҖбһ¶бһҡбһҖбһҺбҹ’бһҸбһ¶бһӣбһ“бҹ… San Francisco, California бһ бһҫбһҷбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһҮбһ¶бһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһҹбһ”бҹ’бһ”бһ»бһҡбһҹбһ’бһҳбҹҢвҖӢбһҸбһ¶бһҳбһ…бҹ’бһ”бһ¶бһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ‘бҹҒбһҹбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбҹҒбһҹбҹ’бһӮбһ¶бһӣбҹӢбһ…бҹ’бһҡбһҫбһ“бһҸбһ¶бһҳбһҡбһҷбҹҲвҖӢ Wikipediaбҹ” бһҹбҹ’бһҗбһ¶бһ”бһ“бһ·бһҖбһҡбһ”бһҹбҹӢбһңбһ¶бһӮбһәбһӣбҹ„бһҖ Jimmy Walesбҹ” бһӮбһ¶бһҸбҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһңбһ¶бһ“бҹ… бһҗбҹ’бһ„бҹғбһ‘бһёбҹўбҹ  бһҒбҹӮбһҳбһ·бһҗбһ»бһ“бһ¶ бһҶбҹ’бһ“бһ¶бҹҶбҹўбҹ бҹ бҹЈбҹ” бһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһ“бҹҒбҹҮбһ”бһ¶бһ“бһҡбһёбһҖбһҠбһ»бҹҮбһҠбһ¶бһӣбһҷбҹүбһ¶бһ„бһҶбһ¶бһ”бҹӢбһҡбһ бҹҗбһҹ бһҠбҹ„бһҷбһҖбҹ’бһ“бһ»бһ„бһҶбҹ’бһ“бһ¶бҹҶбҹўбҹ бҹўбҹў бһңбһ¶бһҳбһ¶бһ“бһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбһ”бһҳбҹ’бһҡбһҫбһҖбһ¶бһҡбһ“бһ·бһ„бһўбҹ’бһ“бһҖбһ…бһјбһӣбһҡбһҪбһҳбһ–бһёбһҹбҹ’бһҗбһ¶бһ”бҹҗбһ“бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҹбһҡбһ»бһ”бһҮбһ¶бһ„бҹ§бҹ бҹ бһ“бһ¶бһҖбҹӢбҹ” бһ…бҹҶбһ–бҹ„бҹҮбһ…бҹҶбһҺбһјбһӣбһҮбһ¶бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҶбҹ’бһ“бһ¶бҹҶбһҳбһ¶бһ“бһҒбҹ’бһ‘бһ„бҹӢбһҮбһ¶бһ„ бҹЎбҹҘбҹҘбһӣбһ¶бһ“бһҠбһ»бһӣбҹ’бһӣбһ¶бһўбһ¶бһҳбҹҒбһҡбһ·бһҖ бһ бһҫбһҷбһҒбҹ’бһ‘бһ„бҹӢбһ…бҹҶбһҺбһ¶бһҷбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҶбҹ’бһ“бһ¶бҹҶбһҮбһ¶бһ„ бҹЎбҹӨбҹҰбһӣбһ¶бһ“бһҠбһ»бһӣбҹ’бһӣбһ¶бһўбһ¶бһҳбҹҒбһҡбһ·бһҖбһ•бһ„бһҠбҹӮбһҡбҹ” бһ”бһ“бҹ’бһҸбһ‘бҹ…бһ•бҹ’бһ“бҹӮбһҖбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ бһҳбһ¶бһ“бһҠбһјбһ…бһҮбһ¶бҹ–

* DNS (бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҲбҹ’бһҳбҹ„бҹҮбһңбҹӮбһ”бһҹбһ¶бһҷ) бһ”бҹ’бһҡбһҫ gdnsd (бһҮбһ¶бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ”бһҳбҹ’бһҡбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҲбҹ’бһҳбҹ„бҹҮбһңбҹӮбһ”бһҹбһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһ”бҹҶбһҺбһ„бһҡбһҖбҹ’бһҹбһ¶бһҹбҹ’бһҗбҹҒбһҡбһ—бһ¶бһ–бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҸбһ¶бһҳбһҸбҹҶбһ”бһ“бҹӢбһ—бһјбһҳбһ·бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ)
* бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ”бһҳбҹ’бһҡбһҫбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһӮбһә Debian GNU/Linux
* бһңбҹӮбһ”бһҹбһ¶бһҷбһ…бһҳбҹ’бһ”бһ„бһӮбһә MediaWiki бһҹбһҡбһҹбҹҒбһҡбһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶ PHP (~70%) бһ“бһ·бһ„ JavaScript (~30%)
* бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһҡбһҪбһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбҹ’бһҸбһ»бһҖбһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„ MariaDB бһҸбһ¶бҹҶбһ„бһ–бһёбһҶбҹ’бһ“бһ¶бҹҶбҹўбҹ бҹЎбҹЈ
* бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҳбһ¶бһ“бҹ– бһ”бһёбһ“бҹ…бһўбһ¶бһҳбҹҒбһҡбһ·бһҖ (eqiad, codfw, ulsfo) бһ–бһёбһҡбһ“бҹ…бһўбҹҠбһәбһҡбҹүбһ»бһ” (esams) бһ“бһ·бһ„ бһҳбһҪбһҷбһ‘бҹҖбһҸбһ“бҹ…бһўбһ¶бһҹбҹҠбһё (drmrs)
