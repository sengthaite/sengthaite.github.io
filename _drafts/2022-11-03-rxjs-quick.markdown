# RxJS Quick

* Composing **asynchronous** and **event-based programs** using observable sequences.
* ReactiveX => Observer pattern + Iterator pattern + functional programming with collections.
* Essential concepts: **Observable**, **Observer**, **Subscription**, **Operators**, **Subject**, **Schedulers**
